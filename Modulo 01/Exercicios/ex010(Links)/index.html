Usando links externos

Para adicionar link externos para um site usamos a tag A e colocamos a URL no atributo HREF

Voce pode acessar o meu repositorio publico no GitHub

Usando apenas a ta A, ao clicar no link o usuario sera redirecionado para outra pagina e a pagina inicial sera fechada, para abrir uma pagina manterndo a pagina inicial aberta usamos o atributo TARGET="". Esse atributo possui varios valores, sendo eles:

  • _blank vai abrir o link em uma nova janela em branco.
  • _self vai abrir o link na janela ou frame atual(padrao).
  • _top vai desfazer todos os frames e abrir o destino no navegador completo.
  • _parent similar ao uso do _top em uma referencia a janela mae.
  • nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir.

podemos tambem usar o atributo REL="" para indica a natureza do destino, esse atributo tambem possui alguns valores, sao eles:

  • next indica que o link e para proxima parte do documento atual.
  • prev indica que o link e para parte anterior do documento atual.
  • author indica que e um link para o site do autor do artigo atual.
  • external indica que e um link para outro site que nao faz parte do link atual (site externo).
