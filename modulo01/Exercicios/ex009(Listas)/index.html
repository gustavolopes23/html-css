Trabalhando com listas

Listas Ordenadas

Para criar uma lista ordenada vamos usar a tag OL(ordered listis) e para delimitar a lista a tag LI no começo de cada item da lista

  1. Acordar
  2. Fazer uma ligaçao
  3. Tomar cafe
  4. Escovar os dentes
  5. Estudar
  6. Almocar
  7. Ir para o trabalho
  8. Volta para casa
  9. Jantar
  10. Dormir

A tag OL possui o parametro TYPE para configurar o tipo de marcador da lista

  • type="1" ex. 1, 2, 3, 4
  • type="A" ex. A, B, C, D
  • type="a" ex. a, b, c, d
  • type="I" ex. I, II, III
  • type="i" ex. i, ii, iii

Listas nao ordenadas

para as listas nao ordenadas trocamos a tag para UL e para delimitar cada item da lista usamos LI tambem

  • Pao
  • Leite
  • Arroz
  • Feijao

Os parametros TYPE de uma lista nao ordenada sao

  • DISC= padro, uma bolinha preta totalmente pintada
  • CIRCLE= Uma bola com a borda preta e sem preenchimento
  • SQUARE= Um pequeno quadrado preto totalmente pintado
