Tags usadas

  • <picture>
  • <img>
  • <source>

a tag SOURCE possui tres atributos

  • type: vai indicar o media type da imagem que usamos
  • srcset: vai configurar o nome da imagem que sera carregada quando o tamanho indicado for atingido
  • media: indica o tamanho maximo a ser considerado para carregar a imagem indicado no atributo srcset
